# Automation testing of a Telnyx website with Playwright and Allure Reporter


## Versions
- Node JS - 18.12.1
- NPM - 8.19.3
- playwright - 1.31.1


## How to run on your local machine
1. Clone this repo by using command in ```Git```.
```
git clone https://github.com/ddpolianskyi/telnyx-playwright.git
```
2. Install all requirement packages by using command in ```cmd```.
```
npm install
```
3. Run tests by using command:
```
npm run test
```
4. Generate Allure HTML report:
```
npm run allure:generate
```
5. Open Allure HTML report:
```
npm run allure:open
```
